Handling missing values

  • The first way used to handle missing values was to use the median impute method: medianImpute.

  • The second way used to handle missing values was to use the knn method: knnImpute.

  • The third way used to handle missing values was to use the bagging method: bagImpute.

Decision Trees Performance on Imputation Methods

  • Median Impute proved to have the best accuracy. Used median imputed data going forward.
  • Encoding the categorical variables improved accuracy and cp of the decision tree.
  • Accuracy was .827 CP was .01355
  • Encoded data produced accuracy of .829 and CP of .0087

Model Results

  • Decision Tree Tuned: Accuracy of .86 and CP held constant at .005
  • Random Forest: The final values used for the model were mtry = 19, splitrule = gini and min.node.size = 1. Accuracy of .9196 (Champion Model)
  • GLM: The final values used for the model were alpha = 0.1 and lambda = 0.000597716. Accuracy of .862
  • Neural Network: The final values used for the model were size = 5 and decay = 0.1. Accuracy of .852
  • Neural Network Tuned: Tuning parameter 'size' was held constant at a value of 3 Tuning parameter 'decay' was held constant at a value of 0.2. Accuracy of .852
